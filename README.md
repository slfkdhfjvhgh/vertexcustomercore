# Vertex Air Customer Core

Railway variables:
- DISCORD_TOKEN
- GUILD_ID
- OPERATIONS_ROLE_ID
- TIMEZONE=Asia/Kolkata

Enable Server Members Intent and Message Content Intent.

Commands:
- .reply <message>
- .areply <message>
- .openfor <user_id>
- .ban <user_id>
- .unban <user_id>
- .close
- ;schedule

The automatic server unlock, boarding, and server lock announcements are not included yet because their exact timings, channels, and message formats have not been supplied.

- `;unlock <flight number>`
- `;lock <flight number>`
